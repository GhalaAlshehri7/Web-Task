# Web Task

## Description
This project is a simple web application built using HTML, CSS, JavaScript, PHP, and MySQL.

The application allows users to enter their name and age, save the information in a MySQL database, display the saved data in a table, and change the user status using a Toggle button.

## Features

- Add a new user (Name and Age)
- Save data to MySQL database
- Display saved records in a table
- Toggle user status between 0 and 1
- Update the data without manually refreshing the page

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Project Files

- index.php
- db.php
- add.php
- toggle.php
- script.js
- style.css
- database.sql

## How to Run

1. Create a MySQL database.
2. Import the `database.sql` file.
3. Update the database information in `db.php`.
4. Upload the project files to your web server.
5. Open the website in your browser.
