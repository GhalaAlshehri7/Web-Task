# Simple Web Task

## Description
This project is a simple web application built using:

- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Features

- Add a new user
- Save data to MySQL
- Display users in a table
- Toggle status between 0 and 1
- Update without manually refreshing the page
